AGGIORNAMENTO PROVA CARICO PALO

Copia la cartella src dentro la tua cartella Git corretta:
C:\Users\larac\Downloads\prova-carico-pali-v2\prova-carico-pali

Quando Windows chiede se vuoi sostituire i file, scegli Sostituisci.

Poi apri PowerShell nella cartella Git e lancia:

npm run build
git status
git add .
git commit -m "Automatizza bar prova carico pali"
git push origin main

Cosa fa l'aggiornamento:
- 100% = carico di esercizio/SLE
- 150% = carico massimo di collaudo
- bar automatici = kN del gradino / coefficiente taratura kN/bar
- il perito compila solo i 3 comparatori
- PDF e CSV riportano i bar automatici
- grafico unico carico/cedimento
